**Context** :

This dataset shows customer brief information, historical use of discount or BOGO (Buy One Get One) promotions, offers that have been made, and the conversion result (buy or not). The conversion average value = $25."

**Methodology** :

1. Data Profiling
2. Exploratory Data Analysis (EDA)
3. Uplift Modeling
4. Model Evaluation
5. Shapley Additive explaanations (SHAP)
